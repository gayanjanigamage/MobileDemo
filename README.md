## Pre-requisites
* Java
* Maven
* Appium
* Android

## Setting up environment
You can set Environment Variables in your computer. In System Variable section add these variables and relevant values.

* JAVA_HOME
* MAVEN_HOME
* ANDROID_HOME

##Running locally
* mvn clean
* mvn install

Appium server will now be running on port 4723,
